## Instructions

- Write in `{document_output_language}`
- Goals are visionary and aspirational — never metrics or tactics
- Objectives are SMART: Specific, Measurable, Achievable, Relevant, Time-bound
- Standard structure is 3 goals × 3 objectives — use 4–5 objectives only when genuinely necessary
- Order goals hierarchically: primary outcome goal first, prerequisite goals below
- After writing, confirm path to Saga so she can continue to the archetype workshops
