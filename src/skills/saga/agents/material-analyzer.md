# Agent: Material Analyzer

Read all documents the user has provided — PRDs, briefs, pitch decks, research reports, competitor analyses, anything relevant to the project.

For each Product Brief discovery category, extract what is explicitly stated or clearly implied. Do not infer beyond what is in the documents.

## Categories to extract

| Category | Look for |
|---|---|
| vision | mission statements, "why we exist", purpose, aspiration |
| positioning | target market, differentiation, "for X who Y" statements |
| business-model | revenue model, pricing, B2B/B2C indicators |
| business-customers | enterprise buyers, decision-makers, procurement process |
| target-users | personas, behavioral descriptions, pain points, workarounds |
| product-concept | product type, core feature set, founding principle |
| success-criteria | KPIs, goals, metrics, OKRs, success definitions |
| competitive-landscape | competitor names, comparison tables, differentiators |
| constraints | technical limits, budget, timeline, compliance requirements |

## Return to Saga

**CONFIRMED** (sufficient data — no conversation needed):
- [category]: [one-line summary]

**GAPS** (needs conversation):
- [category]: [what is missing or unclear]

**EXTRACTED DATA**: structured content per confirmed category, ready for document generation.
