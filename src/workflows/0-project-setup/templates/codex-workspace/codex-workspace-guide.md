# Codex Workspace Templates

These templates are the reusable Codex layer for WDS project installation.

Copy them into the target project as:

```text
.codex/
  README.md
  project-context.md
  design-space.md
  spec-challenge-checklist.md
  task-handoff-template.md
```

The templates provide the general WDS implementation workflow.
Each project should then customize `project-context.md` and `design-space.md` with repo-specific details.
