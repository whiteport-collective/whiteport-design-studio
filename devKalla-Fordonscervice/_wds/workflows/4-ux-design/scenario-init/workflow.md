# Scenario Initialization Workflow

**Goal:** Transform Trigger Map insights into a clear scenario foundation for sketching

**Your Role:** Conversational facilitator helping designers identify the core feature, natural starting point, mental state, and shortest path to mutual success.

---

## WORKFLOW ARCHITECTURE

This uses **step-file architecture** for disciplined execution:

### Core Principles

- **Conversational Discovery**: Question-based approach to uncover insights
- **Trigger Map Connection**: Links strategic foundation to first design
- **Shortest Path Focus**: Identifies minimum viable journey to mutual success
- **Mental State Awareness**: Captures user's emotional context
- **Strategic Foundation**: Creates Value Trigger Chain before sketching

## THE 7-STEP PROCESS

### 1. Platform Strategy Confirmation
Inherit and confirm platform strategy from Product Brief

### 2. Feature Selection
What core feature delivers value to primary target group?

### 3. Entry Point
Where does user first encounter the solution?

### 4. Mental State
How are they feeling at this moment?

### 5. Mutual Success
What does winning look like for business and user?

### 6. Shortest Path
Minimum steps from starting point to mutual success

### 7. Value Trigger Chain
Crystallize scenario strategy before sketching
