# Module 19: Agent Coordination

**Time: 20 min | Advanced | Optional**

---

## Beyond Single Sessions

You've learned to work with one agent at a time. Saga for strategy. Freya for design. Mimir for building. Each session ends, knowledge disappears, the next agent starts fresh.

WDS solves this through the `_progress/` folder — a built-in coordination layer that every agent reads and writes automatically.

This module goes deeper on that system: what good handoffs contain, how to troubleshoot broken chains, and how to run confident multi-agent projects without losing context.

---

### Lesson 1: The `_progress/` System

How WDS agents hand off work through files. The wrap, handoff, and start workflow. What good handoffs contain and why they work.
